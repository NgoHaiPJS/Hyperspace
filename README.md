# Hyperspace — Space Themed Button

This small web demo shows a dark purple space theme with a centered "Engage Warp" button. Pressing the button triggers a short "warp" animation where stars accelerate and a soft hum plays (if your browser allows audio).

Files:
- `index.html` — main page
- `styles.css` — styling and theme
- `script.js` — starfield canvas and interaction

To run locally:
1. Open `index.html` in a modern browser (Chrome, Edge, Firefox).
2. Click the `Engage Warp` button in the center.

Note: Browsers often require a user gesture to enable audio; clicking the button satisfies that requirement.
